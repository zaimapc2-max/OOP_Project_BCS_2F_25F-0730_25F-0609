♟️ OOP Chess Game (C++)

A console-based Chess game developed using Object-Oriented Programming (OOP) principles in C++. This project was built as a final project for our 2nd semester to demonstrate core OOP concepts such as encapsulation, inheritance, polymorphism, and abstraction.
Authors:
Zaima Amjad (25F-0609)
Saba Tahir (25F-0730)
Features
♜ Complete Chess board implementation (8x8)
♞ All standard Chess pieces:
Pawn
Rook
Knight
Bishop
Queen
King
Turn-based gameplay (Player vs Player)
Move validation according to Chess rules
Piece capturing system
Illegal move prevention
Colored console UI using ANSI escape codes
Continuous game loop until checkmate or exit

OOP Concepts Used:
This project strongly follows OOP design principles:

Encapsulation → Each piece manages its own data and behavior
Inheritance → Base class Piece with derived classes (Pawn, Rook, etc.)
Polymorphism → Virtual functions for movement logic
Abstraction → Game logic separated from display and piece behavior
